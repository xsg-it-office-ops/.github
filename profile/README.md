# IT Office Ops Repository

> 這是直屬 IT 處的輔助開發小隊倉庫。
> 本倉庫為多專案整合與協作空間，並非單一系統專案。

---

## 倉庫定位

本倉庫用於支援 IT 處內部的開發、整合與維運工作，涵蓋：

- 多服務後端模組
- 多前端應用
- 資料與腳本資源
- 基礎設施與部署設定

本倉庫目標是提供可持續、可擴充、可協作的技術支援能力，快速回應處內行政與業務需求。
